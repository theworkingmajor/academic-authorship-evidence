# academic-authorship-evidence
Research notes and conceptual frameworks on authorship verification in academic writing, including limitations of AI text classifiers, false positives, and writing-process evidence. No production code.

This repository collects research notes, references, and conceptual frameworks related to authorship verification in academic writing.
Focus areas:
- Limitations of AI-text classifiers
- False positives and bias
- Writing-process metadata
- Evidence-based academic integrity

This repository is research-oriented and does not include production code.
